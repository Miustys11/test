## 概要

PHP/laravelの勉強として作成した、ECサイトのポートフォリオです。 一般的なメECサイトに実装される、基本的な機能を実装しました。

## バージョン

PHP 7.3.11<br>
laravel 5.8.38<br>

## 実装機能

### 商品関連

・一覧表示機能<br>
・詳細表示機能<br>
・編集機能<br>
・削除機能<br>
・検索機能<br>
・カート機能<br>
・ランキング機能<br>
・お気に入り機能<br>
・リコメンド機能<br>
・決済機能<br>

### ユーザー関連

・認証機能

## その他

AWSによる環境構築<br>
データベースにはMySQLを使用<br>
Herokuへのデプロイ<br>
